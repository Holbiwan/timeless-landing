About Timeless

I started building Timeless because I wanted to learn Flutter and Firebase hands-on. As a student myself, I know how frustrating job searching can be with all the different platforms and complicated forms. So I'm trying to make something simpler and more straightforward.

This project is a Portfolio Project for Holberton School, where I'm learning software engineering. It represents my journey in mobile development and my passion for creating user-friendly applications.
